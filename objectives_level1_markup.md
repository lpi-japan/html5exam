## 認定条件 <!-- omit in toc -->

HTML/CSS Professional Certification Level 1 試験に合格すると HTML/CSS Professional Certification Level 1 に認定されます。

## 望まれるスキルレベル <!-- omit in toc -->

- HTML5 (HTML Standard, CSS などの Web 技術) を使って Web コンテンツを制作できる。
- ユーザ体験を考慮した Web コンテンツを設計・制作できる。
- スマートフォンや組み込み機器など、ブラウザが利用可能な様々なデバイスに対応したコンテンツを制作できる。
- HTML5 (HTML Standard, CSS などの Web 技術) で何ができるか、どういった技術を使うべきかの広範囲の基礎知識を有する。

# HTML/CSS Level 1 出題範囲 <!-- omit in toc -->

## 1. HTML

### 1.1. HTMLの基礎と書式

**出題種別**
- 知識問題
- コードリーティング問題
- 記述問題

**説明（望まれるスキル）**
- HTML標準に沿って正しくマークアップできる。

**主要な知識範囲**
- 文書型宣言、言語コードや文字コードの指定
- 文字参照、リンク要素、メタ要素
- 正しい書式とセマンティクスの基礎
- Web関連のスクリプトやセキュリティ対策の概要

**重要な技術要素**
- DOCTYPE宣言
- html要素(lang属性, meta charsetなど)
- 文字参照(&nbsp;, &amp; など)
- DOMの基礎・概念
- カスタムデータ属性

### 1.2. HTTP, HTTPSプロトコル
**出題種別**
- 知識問題
- 記述問題

**説明（望まれるスキル）**
- HTTPやHTTPSプロトコルの基礎を理解し、ステータスコードやヘッダ情報を把握して適切に利用できる。

**主要な知識範囲**
- ブラウザとWebサーバ間の通信手順
- HTTPリクエストメソッドの種類と違い
- ステータスコードやエラーレスポンスの意味
- HTTPS, SSL/TLSの基礎

**重要な技術要素**
- HTTP/1.1, HTTP/2
- ステータスコード(200, 301, 404など)
- 認証方式(Basic, Digestなど)
- Cookie, セキュリティ関連ヘッダ

### 1.3. Web関連技術の概要
**出題種別**
- 知識問題
- 記述問題

**説明（望まれるスキル）**
- Web関連のスクリプトやセキュリティ対策などの概要を理解し、チームで円滑にコミュニケーションできる。

**主要な知識範囲**
- Ajax, セッション, 画像フォーマットの概要
- SQLインジェクション, XSSなどのセキュリティ脅威

**重要な技術要素**
- DOMの基礎・概念
- カスタムデータ属性
- MVCアーキテクチャ

### 1.4. Emojiの使用

**出題種別**

- 知識問題
- コードリーティング問題
- 記述問題

**説明（望まれるスキル）**

- HTMLでのemojiの基本的な使用方法を理解し、Webコンテンツに組み込むことができる。

**主要な知識範囲**

- emojiの基本概念
- emojiのコードポイント

**重要な技術要素**

- Unicodeとemoji
- `&#x1F600;` などのコードポイント

## 2. CSS

### 2.1. スタイルシートの基本

**説明（望まれるスキル）**
- CSSの基本構造や文法を理解し、適切なセレクタやプロパティを活用できる。

**主要な知識範囲**
- CSSの基本構造と文法
- セレクタとプロパティの概念
- 外部スタイルシートのリンク方法

**重要な技術要素**
- セレクタの種類（タグ、クラス、IDセレクタなど）
- プロパティと値のペア
- インラインスタイル、内部スタイルシート、外部スタイルシート

### 2.2. CSSデザイン

**説明（望まれるスキル）**
- レイアウトや配色などの基礎を整理し、視覚的にわかりやすいデザインを行える。

**主要な知識範囲**
- レイアウトの基本
- 色と背景の設定
- テキストのスタイリング

**重要な技術要素**
- ボックスモデル（margin, border, padding, content）
- フロートとクリア
- フォントプロパティとテキストプロパティ

### 2.3. カスケード（優先順位）

**説明（望まれるスキル）**
- カスケードや継承、特異性を正しく理解し、効率的にスタイルを管理できる。

**主要な知識範囲**
- カスケードの概念と仕組み
- 継承の基本
- 特異性（Specificity）の計算方法

**重要な技術要素**
- !importantの使用
- 継承されるプロパティとされないプロパティ
- セレクタの特異性ルール

### 2.4. Flexboxの基本

**説明（望まれるスキル）**
- Flexboxを用いて柔軟なレイアウトが組める基礎を身につける。

**主要な知識範囲**
- Flexboxを使ったシンプルなレイアウト作成

**重要な技術要素**
- display: flex
- flex-direction
- justify-content
- align-items

## 3. 要素

### 3.1. メディア要素

**説明（望まれるスキル）**
- オーディオやビデオなどの活用方法を理解し、メディア要素を効果的に埋め込める。

**主要な知識範囲**
- オーディオとビデオの基本
- メディア要素の属性とメソッド
- メディアの埋め込み方法

**重要な技術要素**
- audio要素
- video要素
- source要素
- track要素

### 3.2. インタラクティブ要素

**説明（望まれるスキル）**
- インタラクティブ要素を適切に設定して、ユーザーの操作性を向上できる。

**主要な知識範囲**
- インタラクティブ要素の種類と使用方法
- フォーム要素の基本
- ユーザーインタラクションの向上

**重要な技術要素**
- button, input, select, textarea, label要素
- form要素とその属性
- datalist要素
- progress要素とmeter要素

### 3.3. 基本的なARIAの利用

**説明（望まれるスキル）**
- シンプルなARIA属性を用いてアクセシビリティに配慮した実装ができる。

**主要な知識範囲**
- 画面読み上げ等を意識したシンプルなARIA属性の使用方法

**重要な技術要素**
- aria-hidden, aria-label, role属性

## 4. レスポンシブWebデザイン

### 4.1. マルチデバイス対応

**説明（望まれるスキル）**
- レスポンシブデザインの基本を理解し、各デバイスに応じたレイアウトを実装できる。

**主要な知識範囲**
- レスポンシブデザインの基本原則
- メディアクエリの使用方法
- フレキシブルグリッドレイアウト
- ビューポートの設定
- モバイルファーストデザイン

**重要な技術要素**
- CSSメディアクエリ
- フレキシブルボックス（Flexbox）
- CSSグリッドレイアウト
- ビューポートメタタグ
- 画像のレスポンシブ対応（srcset属性、picture要素）

### 4.2. メディアクエリ

**説明（望まれるスキル）**
- メディアクエリを駆使してデバイス幅に応じたスタイルを切り替えられる。

**主要な知識範囲**
- メディアクエリの基本
- メディアクエリの構文と使用方法
- ブレークポイントの設定

**重要な技術要素**
- @mediaルール
- min-width, max-width, min-height, max-heightなどのメディア特性
- 複数の条件を組み合わせたメディアクエリ
