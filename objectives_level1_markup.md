# HTML/CSS Level 1 概要

## 名称

|                    |                                               |
|--------------------|-----------------------------------------------|
| **認定名**         | HTML/CSS Professional Certification Level 1   |
| **試験の正式名称** | HTML/CSS Professional Certification Level 1 Exam |

## 認定条件

HTML/CSS Professional Certification Level 1 試験に合格すると HTML/CSS Professional Certification Level 1 に認定されます。

## 望まれるスキルレベル

- HTML5 (HTML Standard, CSS などの Web 技術) を使って Web コンテンツを制作できる。
- ユーザ体験を考慮した Web コンテンツを設計・制作できる。
- スマートフォンや組み込み機器など、ブラウザが利用可能な様々なデバイスに対応したコンテンツを制作できる。
- HTML5 (HTML Standard, CSS などの Web 技術) で何ができるか、どういった技術を使うべきかの広範囲の基礎知識を有する。

# HTML/CSS Level 1 出題範囲

## *1.1 HTML*

### 1.1.1 HTMLの基本

### 主要な知識範囲
- HTMLの基本構造
- 要素とタグの概念
- HTML文書の基本的な構成要素

### 重要な技術要素
- <!DOCTYPE>宣言
- html, head, body要素
- meta, title, link要素

### 1.1.1 HTTP, HTTPSプロトコル
**出題種別**
- 知識問題
- 記述問題

**説明（望まれるスキル）**
- HTTPやHTTPSプロトコルの基礎を理解し、ステータスコードやヘッダ情報を把握して適切に利用できる。

**主要な知識範囲**
- ブラウザとWebサーバ間の通信手順
- HTTPリクエストメソッドの種類と違い
- ステータスコードやエラーレスポンスの意味
- HTTPS, SSL/TLSの基礎

**重要な技術要素**
- HTTP/1.1, HTTP/2
- ステータスコード(200, 301, 404など)
- 認証方式(Basic, Digestなど)
- Cookie, セキュリティ関連ヘッダ

### 1.1.2 HTMLの書式
**出題種別**
- 知識問題
- コードリーティング問題
- 記述問題

**説明（望まれるスキル）**
- HTML標準に沿って正しくマークアップできる。

**主要な知識範囲**
- 文書型宣言, 言語コードや文字コードの指定
- 文字参照, リンク要素, メタ要素に関する記述方法

**重要な技術要素**
- <!DOCTYPE html>
- html要素(lang属性, meta charsetなど)
- 文字参照(&nbsp;, &amp; など)

### 1.1.3 Web関連技術の概要
**出題種別**
- 知識問題
- 記述問題

**説明（望まれるスキル）**
- Web関連のスクリプトやセキュリティ対策などの概要を理解し、チームで円滑にコミュニケーションできる。

**主要な知識範囲**
- Ajax, セッション, 画像フォーマットの概要
- SQLインジェクション, XSSなどのセキュリティ脅威

**重要な技術要素**
- DOM
- カスタムデータ属性
- MVCアーキテクチャ
