## 望まれるスキルレベル <!-- omit in toc -->

- HTML5 (HTML Standard, CSS などの Web 技術) を使って Web コンテンツを制作できる。
- ユーザ体験を考慮した Web コンテンツを設計・制作できる。
- スマートフォンや組み込み機器など、ブラウザが利用可能な様々なデバイスに対応したコンテンツを制作できる。
- HTML5 (HTML Standard, CSS などの Web 技術) で何ができるか、どういった技術を使うべきかの広範囲の基礎知識を有する。

# HTML/CSS Level 1 出題範囲

## 1. HTML

### 1.1. HTMLの基本

**主要な知識範囲**
- HTMLの基本構造
- 要素とタグの概念
- HTML文書の基本的な構成要素

**重要な技術要素**
- <!DOCTYPE>宣言
- html, head, body要素
- meta, title, link要素

### 1.2. HTTP, HTTPSプロトコル
**出題種別**
- 知識問題
- 記述問題

**説明（望まれるスキル）**
- HTTPやHTTPSプロトコルの基礎を理解し、ステータスコードやヘッダ情報を把握して適切に利用できる。

**主要な知識範囲**
- ブラウザとWebサーバ間の通信手順
- HTTPリクエストメソッドの種類と違い
- ステータスコードやエラーレスポンスの意味
- HTTPS, SSL/TLSの基礎

**重要な技術要素**
- HTTP/1.1, HTTP/2
- ステータスコード(200, 301, 404など)
- 認証方式(Basic, Digestなど)
- Cookie, セキュリティ関連ヘッダ

### 1.3. HTMLの書式
**出題種別**
- 知識問題
- コードリーティング問題
- 記述問題

**説明（望まれるスキル）**
- HTML標準に沿って正しくマークアップできる。

**主要な知識範囲**
- 文書型宣言, 言語コードや文字コードの指定
- 文字参照, リンク要素, メタ要素に関する記述方法

**重要な技術要素**
- DOCTYPE宣言
- html要素(lang属性, meta charsetなど)
- 文字参照(&nbsp;, &amp; など)

### 1.4. Web関連技術の概要
**出題種別**
- 知識問題
- 記述問題

**説明（望まれるスキル）**
- Web関連のスクリプトやセキュリティ対策などの概要を理解し、チームで円滑にコミュニケーションできる。

**主要な知識範囲**
- Ajax, セッション, 画像フォーマットの概要
- SQLインジェクション, XSSなどのセキュリティ脅威

**重要な技術要素**
- DOM
- カスタムデータ属性
- MVCアーキテクチャ

## 2. CSS

### 2.1. スタイルシートの基本

**主要な知識範囲**
- CSSの基本構造と文法
- セレクタとプロパティの概念
- 外部スタイルシートのリンク方法

**重要な技術要素**
- セレクタの種類（タグ、クラス、IDセレクタなど）
- プロパティと値のペア
- インラインスタイル、内部スタイルシート、外部スタイルシート

### 2.2. CSSデザイン

**主要な知識範囲**
- レイアウトの基本
- 色と背景の設定
- テキストのスタイリング

**重要な技術要素**
- ボックスモデル（margin, border, padding, content）
- フロートとクリア
- フォントプロパティとテキストプロパティ

### 2.3. カスケード（優先順位）

**主要な知識範囲**
- カスケードの概念と仕組み
- 継承の基本
- 特異性（Specificity）の計算方法

**重要な技術要素**
- !importantの使用
- 継承されるプロパティとされないプロパティ
- セレクタの特異性ルール

### 2.4. Flexboxの基本

**主要な知識範囲**
- Flexboxを使ったシンプルなレイアウト作成

**重要な技術要素**
- display: flex
- flex-direction
- justify-content
- align-items

## 3. 要素

### 3.1. 要素と属性の意味（セマンティクス）

**主要な知識範囲**
- HTML要素の意味と役割
- セマンティックHTMLの重要性
- アクセシビリティの向上

**重要な技術要素**
- header, footer, nav, article, section, aside, mainなどのセマンティック要素
- aria属性
- role属性

### 3.2. メディア要素

**主要な知識範囲**
- オーディオとビデオの基本
- メディア要素の属性とメソッド
- メディアの埋め込み方法

**重要な技術要素**
- audio要素
- video要素
- source要素
- track要素

### 3.3. インタラクティブ要素

**主要な知識範囲**
- インタラクティブ要素の種類と使用方法
- フォーム要素の基本
- ユーザーインタラクションの向上

**重要な技術要素**
- button, input, select, textarea, label要素
- form要素とその属性
- datalist要素
- progress要素とmeter要素

### 3.4. 基本的なARIAの利用
**主要な知識範囲**
- 画面読み上げ等を意識したシンプルなARIA属性の使用方法

**重要な技術要素**
- aria-hidden, aria-label, role属性

## 4. レスポンシブWebデザイン

### 4.1. マルチデバイス対応

**主要な知識範囲**
- レスポンシブデザインの基本原則
- メディアクエリの使用方法
- フレキシブルグリッドレイアウト
- ビューポートの設定
- モバイルファーストデザイン

**重要な技術要素**
- CSSメディアクエリ
- フレキシブルボックス（Flexbox）
- CSSグリッドレイアウト
- ビューポートメタタグ
- 画像のレスポンシブ対応（srcset属性、picture要素）

### 4.2. メディアクエリ

**主要な知識範囲**
- メディアクエリの基本
- メディアクエリの構文と使用方法
- ブレークポイントの設定

**重要な技術要素**
- @mediaルール
- min-width, max-width, min-height, max-heightなどのメディア特性
- 複数の条件を組み合わせたメディアクエリ
